HTML
HTML (HyperText Markup Language) is the standard language used to create and structure web pages by defining elements like headings, paragraphs, links, images, and other content.
markup language
A markup language is a system for annotating text in a way that defines its structure and presentation, typically using tags or symbols.
DOCTYPE
DOCTYPE (Document Type Declaration) is an instruction at the beginning of an HTML document that tells the web browser which version of HTML to use. It ensures the page is rendered correctly.
required HTML tags
The essential HTML tags required for a basic web page structure are: html,head,title,meta and body
head tags
The head section in an HTML document contains metadata and resources for the webpage.
body tags
The body tag in HTML contains all the visible content of a webpage, including text, images, links, and interactive elements.
wireframe HTML tags
Wireframing in HTML refers to creating a basic structural layout of a webpage without styling or design.
semantic
Semantic HTML refers to the use of meaningful HTML tags that clearly describe the structure and purpose of web content. These tags improve accessibility, SEO, and code readability.
paired tags
Paired tags are HTML elements that come in opening (tag) and closing (/tag) pairs to define the start and end of content.
syntax for a closing tag
A closing tag in HTML is used to mark the end of an element and is written with a forward slash (/) before the tag name
void tag
A void tag (also known as a self-closing tag) is an HTML element that does not have a closing tag. It is used to represent elements that don't have any content between an opening and closing tag.
attribute
An attribute in HTML provides additional information about an element. It is placed within the opening tag and usually comes in a name-value pair format.
syntax for an attribute value
The syntax for an attribute value in HTML involves placing the value inside quotes (either single ' or double " quotes). The attribute name is followed by an equal sign (=) and the value.
required attributes for anchor, a, tag
The anchor tag in HTML is used to create hyperlinks. While the href attribute is required to define the destination URL, other attributes are optional depending on the desired functionality.
destinations for a link
When creating hyperlinks with the anchor tag, the href attribute defines the destination of the link.
required attributes for img tag
The img tag in HTML is used to embed an image in a webpage. The required attributes are:src and alt
heading tags
Heading tags in HTML are used to define headings or titles in a webpage.h1,h2,h3,h4,h5 and h6 are heading tags.
